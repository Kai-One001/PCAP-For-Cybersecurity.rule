# PCAP-For-Cybersecurity.rule🛡️

> 为防守方提供可自行复现的脱敏 pcap 样本，用于快速制作、验证检测规则。

## 为什么需要这个项目

漏洞爆发越来越快，尤其是 AI 降低了利用门槛后，网络上铺天盖地的 EXP 让防守窗口被极度压缩。  
--但是呢 可能不会是所有的漏洞，精力有限哈哈哈哈

**这个项目希望筑起一座高墙** —— 收集、脱敏并公开各类攻击手法、漏洞利用、渗透行为的真实抓包，让每一位防守工程师都能直接拿去做：

- IDS/IPS 规则开发（Suricata、Snort 等）
- 网络检测能力验证
- 应急响应演练
- 威胁情报分析


**AI 是加速器，但 pcap 是燃料。**
